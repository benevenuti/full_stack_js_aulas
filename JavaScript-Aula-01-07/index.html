Projeto 1 - "ClickCat"

Um cliente ganhou um gatinho de presente e tirou uma foto bem bacana do mesmo, e ele deseja um software onde as pessoas possam dizer se curtiram ou não a foto de seu gatinho. Adivinha quem foi o premiado para desenvolver este revolucionário programa de computador? Você mesmo!

Veja o exemplo final clicando aqui.

Etapas para o desenvolvimento do software:

  • Criar um arquivo HTML contendo Bootstrap e suas dependências. (O starter kit você encontra clicando aqui!)
  • Criar um código JavaScript que, ao clicar no botão "Gostei" ou "Não Gostei" apenas incremente o valor do button clicado. Fácil não!?

Selecionar os objetos que você quer manipular

var btnGostei = document.getElementById('btn-gostei');
var btnNaoGostei = document.getElementById('btn-nao-gostei');
var spanGostei = document.getElementById('span-gostei');
var spanNaoGostei = document.getElementById('span-nao-gostei');

Aproveite para ler um pouco sobre seletores!

Criar a ação do botão de "Gostei"

btnGostei.onclick = function(){
    console.log('Button Gostei Pressionado!');

    var valorGostei = parseInt(spanGostei.innerText); // pega o conteudo da span
    valorGostei = valorGostei + 1; // adiciona mais um

    console.log(valorGostei); // printa o valor
    spanGostei.innerText = valorGostei;
};

Criar a ação do botão de "Não Gostei"

btnNaoGostei.onclick = function(){
    console.log('Button Não Gostei Pressionado!');

    var valorNaoGostei = parseInt(spanNaoGostei.innerText); // pega o conteudo da span
    valorNaoGostei = valorNaoGostei + 1; // adiciona mais um

    console.log(valorNaoGostei); // printa o valor
    spanNaoGostei.innerText = valorNaoGostei;
}
