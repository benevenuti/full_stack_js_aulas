Projeto 2 - "ClickCat com jQuery"

Vamos mudar o nosso exemplo, criando uma versão do ClickCat usando a biblioteca jQuery para vermos as diferenças.

Veja o exemplo final clicando aqui.

Etapas para o desenvolvimento do software:

  • Precisamos adicionar a lib do jQuery do nosso HTML, isso pode ser feito via CDN ou via tag script.
  • Criar um arquivo HTML contendo Bootstrap e suas dependências. (O starter kit você encontra clicando aqui!)
  • Criar um código JavaScript que, ao clicar no botão "Gostei" ou "Não Gostei" apenas incremente o valor do button clicado. Fácil não!?

Selecionar os objetos que você quer manipular

var btnGostei = $('#btn-gostei');
var btnNaoGostei = $('#btn-nao-gostei');
var spanGostei = $('#span-gostei');
var spanNaoGostei = $('#span-nao-gostei');

Aproveite para ler um pouco sobre seletores usando jQuery!

Criar a ação do botão de "Gostei"

btnGostei.on('click', function(){
    console.log('Button Gostei Pressionado!');

    var valorGostei = parseInt(spanGostei.text()); // pega o conteudo da span
    valorGostei = valorGostei + 1; // adiciona mais um

    console.log(valorGostei); // printa o valor
    spanGostei.text(valorGostei);
});

Criar a ação do botão de "Não Gostei"

btnNaoGostei.on('click', function(){
    console.log('Button Não Gostei Pressionado!');

    var valorNaoGostei = parseInt(spanNaoGostei.text()); // pega o conteudo da span
    valorNaoGostei = valorNaoGostei + 1; // adiciona mais um

    console.log(valorNaoGostei); // printa o valor
    spanNaoGostei.text(valorNaoGostei);
});
